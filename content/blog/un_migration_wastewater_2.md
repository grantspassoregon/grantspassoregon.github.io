# Using the Data Loading Tools

For several years, the Data Loading Tools have been fighting their way into legitimacy. Early adopters had to clone their ArcPro environment and install the third-party _untools_ package to gain access to the toolkit. Today, the Data Loading Tools are a first-class ESRI citizen, and come bundled with ArcPro 3.6. We will be using these tools to read assets from our geometric network and load them into the destination utility network, while preserving key attribute data, and mapping values from existing domains into our target schema.
